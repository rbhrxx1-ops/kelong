# RELAX CAR SERVICE CO. LIMITED. 商业信函档案库

本目录为公司对外商业信函的统一管理档案库，包含品牌规范、信函模版及历史发函记录。所有对外正式函件均须基于本目录的模版生成，以确保企业形象的一致性与专业性。

## 目录文件说明

| 文件名 | 类型 | 说明 |
|---|---|---|
| `Brand_Guidelines_and_Template_Usage.md` | 品牌规范 | 公司信函品牌视觉规范、标准信息及模版使用说明（**必读**） |
| `Relax_Airport_Taxi_Letter_Template.html` | 信函模版 | 公司专用商业信函 HTML 模版，所有新函件均从此文件复制修改 |
| `Klook_API_Integration_Application.md` | 历史函件 | Klook 接送机 API 对接申请函（Markdown 版） |
| `Klook_API_Integration_Application.html` | 历史函件 | Klook 接送机 API 对接申请函（HTML 排版版） |
| `Klook_API_Integration_Application.pdf` | 历史函件 | Klook 接送机 API 对接申请函（最终 PDF 版，可直接发送） |
